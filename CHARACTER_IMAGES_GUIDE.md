# Character Images Guide

## How to Add One Piece Character Images

To use actual One Piece character images that look like the originals:

### 1. Image Requirements
- **Format**: PNG, JPG, or WebP
- **Size**: Recommended 200x200px to 400x400px (square)
- **Style**: Use consistent art style (all official art, all fan art in same style, etc.)
- **Naming**: Use the character slug as filename

### 2. Where to Place Images
Put all character images in:
```
public/images/characters/
```

### 3. Naming Convention
Name files using the character slug:
- `luffy.png` or `luffy.jpg`
- `zoro.png` or `zoro.jpg`
- `nami.png` or `nami.jpg`
- `boa-hancock.png` (or `boahancock.png`)

### 4. Where to Get Images

**Free Options:**
- **One Piece Wiki** - Official character images
- **DeviantArt** - Consistent fan art collections
- **Pinterest** - Curated One Piece character art
- **Official Bandai/Namco sites** - Official artwork

**Tips for Consistency:**
- Use all images from the same source/style
- Or use all official artwork
- Or commission an artist for a consistent custom set
- Or use AI-generated images with consistent prompts

### 6. Testing
After adding images:
1. Refresh your browser
2. Check the markets table - characters should show their images
3. Check the chart - character avatars should appear in the legend

### 7. Fallback
If no image is found for a character, a gradient avatar with the first letter will be shown automatically.
